# Weather Forecast Application

A command-line weather app that fetches real-time data for any city using the OpenWeatherMap API.

## View the app
The full code is in `weather_app.py` above.

## About
Built a Python application that fetches real-time weather data for any city by integrating
with the OpenWeatherMap REST API, handling authentication via API keys.

## Features
- Current weather conditions — temperature, description, humidity, and wind speed
- 5-day forecast showing daily conditions at midday
- Robust error handling for invalid city names and network issues
- Unit conversion — all temperatures in Celsius, wind speed in km/h
- Continuous menu loop so users can check multiple cities without restarting

## Note
You will need a free OpenWeatherMap API key from openweathermap.org.
Replace `YOUR_API_KEY_HERE` in the code with your own key.

## Tech
Python · REST API · OpenWeatherMap API · JSON parsing · Error handling
